# ADIDAS-DATASET-ANALYSIS (TABLEAU)

## Overview
This project presents an interactive Tableau dashboard built using the Adidas Sales dataset. It provides insights into sales performance, profitability, product trends, retailer performance, and regional sales.

## Project Objectives
- Analyze total sales
- Compare regional performance
- Analyze state performance
- Evaluate retailer performance
- Identify top-performing products
- Track sales trends over time

## Tools Used
- Tableau
- Microsoft Excel
- GitHub

## Dataset
Adidas Sales Dataset (public dataset)

## Key Insights
- West region generated the highest sales.
- Men's Street Footwear contributed the highest revenue.
- Sales peaked in July 2021, with the state of Florida leading, and the men's apparel being the highest selling product.
